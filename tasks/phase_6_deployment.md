# Phase 6: Cloud Deployment via Vertex AI Agent Engine

**Goal:** Package and deploy the backend to Vertex AI Agent Engine using `agents-cli`.

**Tasks:**
- [ ] Create and configure `agents-cli-manifest.yaml` for the agent environment.
- [ ] Ensure `agents-cli` is installed and authenticated with Google Cloud.
- [ ] Containerize the FastAPI backend and configure the cloud runtime.
- [ ] Deploy the backend using `agents-cli deploy`.
- [ ] Test the deployed endpoints via the Frontend UI.

**Definition of Done:** The agent system is live and accessible via Google Cloud Agent Runtime, with the frontend successfully interacting with the deployed API.
